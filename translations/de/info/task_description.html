In dieser CheckiO-Mission Roman Numerals müssen Sie eine Dezimalzahl zahl in ihre Darstellung als römische Zahl umwandeln.
Hier müssen Sie dasselbe tun, nur andersherum.

Sie erhalten eine römische Zahl als String und Ihre Aufgabe ist es, diese Zahl in die ihre dezimale Darstellung.

Eine gültige römische Zahl enthält im Rahmen dieser Aufgabe nur römische Ziffern gemäß der folgenden Tabelle enthalten und den Regeln der der subtraktiven Notation.
Sehen Sie sich dies an Wikipedia article für weitere Details zur Bildung römischer Ziffern.
ZiffernWert
I1 (unüblich)
V5 (quinque)
X10 (decem)
L50 (quinquaginta)
C100 (centum)
D500 (quingenti)
M1.000 (mille)

Eingabe: Eine römische Zahl als String.

Ausgabe: Die dezimale Darstellung der römischen Zahl als int.

Beispiel:

{% if interpreter.slug == "js-node" %} 
reverseRoman('VI') == 6
reverseRoman('LXXVI') == 76
reverseRoman('CDXCIX') == 499
reverseRoman('MMMDCCCLXXXVIII') == 3888
{% else %} 
reverse_roman('VI') == 6
reverse_roman('LXXVI') == 76
reverse_roman('CDXCIX') == 499
reverse_roman('MMMDCCCLXXXVIII') == 3888
{% endif %}

Vorbedingung:
len(roman_string) > 0
all(char in "MDCLXVI" for char in roman_string) == True
0 < reverse_roman(roman_string) < 4000
